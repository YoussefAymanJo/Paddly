# Paddly
## Description
Paddly is a customized macro pad that combines both customization and functionality. I actually made this to facilitate my daily use. It it consist of 6 switches with multiple functions, an OLED display for soup, customized 2 rotary encoders, and RGB LEDs. I was a beginner in 3D modeling, but I learned a lot from  Fusion tutorials.

## Why it was built
During my day, I use a laptop  almost all day. But while working on projects or designing on Canva, I struggle and tired alot to use multiplekeysy odoly some features, such as  copy manually and undo. So, built a macro pad that provides all the luxury in my usage. As it has 6 MX Cherry switches with customized functions.And 2 rotary encoders and a soup OLED display.

## How to use it / Build it
First, fabricate the PCB and solder the components. After that, set up the firmware on Pico, kmk firmware. Then test each key, OLED, encoders, and LEDs.after that print 
3d models. After that, assemble 3d  enclosure and setup heatinsert and m3 srews. Finally, test everything again.
